Node.OS by ronnqvist

A lightweight OS-like web interface to bind Node.js apps together, making a 'private cloud'.

Aiming integrate a slick web-interface to launch Node.JS-based apps and for installation of apps through an NPM-based system. The system should be kept light-weight so that it could be run on Linux-based routers and such, as much as possible is to be done in the HTML5-supporting browsers of the users to simplify the server-side code.
